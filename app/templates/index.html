About This Application

This application helps you process quiz grades from Excel or CSV files and convert scores from one scale to another.

Features:

  • Upload Excel or CSV files containing quiz data
  • Convert quiz scores from one scale to another
  • View detailed information about student responses and scores
  • See the converted scores for each student

How to Use:

  1. Click on the "Upload File" button below or in the navigation bar
  2. Select your Excel or CSV file containing quiz data
  3. Enter the original maximum score, new maximum score, and original question value
  4. Submit the form to process the data
  5. View the results showing the converted scores
